Node.JS module: Advanced Logger

Planned feature
---------------
* Custom file logging
* Unix system logger (syslog, logger)
* Logging levels (emerg, alert, crit, etc.)
* Facility (local0 to local7)
* Task runner

Release
-------
**v0.1.0** Support for --version, -v and --help, -h options
